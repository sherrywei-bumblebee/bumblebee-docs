Welcome to Bumblebee Networks!
=====================================

**Bumblebee Networks** is a networking company that provides 
Global Private Link service to everyone on earth. 
It pulls data from the `Open Food Facts database <https://world.openfoodfacts.org/>`_
and offers a *simple* and *intuitive* interface to manage. 

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development, sign up for updates.
